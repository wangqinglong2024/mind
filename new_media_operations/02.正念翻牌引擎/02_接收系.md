# 系列 02 · 接收系（这些将来到你身上）

**一句话定位**：替观众翻开好词——好运、发财、真爱、贵人，一张张金牌翻给你。情绪引擎 = **希望 + 被祝福感**。

## 开场（≈3 秒，越南语配音念完即入词）

- **主题总结大图**打底：把本集好词的意象合成一个震撼场景——**逐集全新设计，禁止跨集复用或模板化**，唯一标准是震撼、吸引、有趣（全片第一冲击点）；
- 品牌四字"正念吸引"+ 越南语小注同场；
- 钩子句三层排版（拼音+中文+越南语），只越南语发音：
  - 通用版："Những điều này sẽ đến với bạn.（这些将来到你身上。）"
  - 年度版："Chúc bạn năm nay nhận đủ những điều này.（愿你今年把这些全部收下。）"
- 合规：chúc/mong 祈愿式（P4），是"送祝福"不是"预言命运"。

## 每词微循环（2.2–2.5 秒）

- **0–0.9s**：金色牌背（带"解"印暗纹）**翻牌** → 翻开即是**越南语词 + 该词动漫图**（冲击音效）——意思先懂、祝福感先到；
- **0.9–1.7s**：**拼音 + 中文毛笔大字**（图上方，鎏金色）砸入 + **中文音色朗读**（咒文时刻）；越南语字不消失；
- **1.7–2.4s**：**收取动效爆发（逐词定制）——词+图化作金光流，被"收进"屏幕下方的福印**（与驱逐系的"毁灭归无"成对称：坏的化为乌有，好的落入你手）；福印每收一词亮一分，四词收满全亮。

## 收取动效：逐词定制（决策人 2026-07-12 拍板：**不做类别固定映射，每个词按词义单独设计**）

系列里只有一个动作是骨架级固定的：**翻牌**（金色牌背 3D 翻转揭词——这是"翻牌子"仪式的本体，牌背统一设计 = 品牌资产）。牌翻开之后，**这个词以什么方式抵达你**，逐词单独设计。设计法则：先问"这个词来到一个人身上，最动人的物理形态是什么？"举例（示范思路，不是映射表）：
- 发财 → 词炸成金币雨落进屏幕下方，叮当入袋
- 好运 → 一尾金色锦鲤驮着词跃入画面，水花化作流光
- 真爱 → 红线从词的笔画里抽出，缠成一个结
- 贵人 → 词后金门缓缓开启，光柱把词推到你面前
- 平安 → 词化作孔明灯升起，再化雨轻轻落下

每个词的动效与音效写进 script.json（fx 描述字段），逐词设计、逐词验收。

工程支撑：与驱逐系共用**动效原语库**（粒子/翻转/光效/物理坠落…参数化组件，只增不删），逐词动效 = 原语组合 + 专属参数 + 专属音效（P6）。

## 示例分镜（年度版 · ≈14.5 秒 · 4 词）
> ⚠️ 本分镜整体仅为示意——主题图、每词配图、每词动效在真实制作时逐集逐词重新设计，不照抄本例。

1. 0–3s｜开场：本集主题大图（逐集设计）+ "正念吸引" + 三语钩子，越南语配音："Chúc bạn năm nay nhận đủ những điều này." + 四张金色牌背排开
2. 3–5.4s｜翻牌 → "May mắn" + 锦鲤跃云海图 → **好运 hǎo yùn** 砸入+中文朗读 → 锦鲤驮词化流光收进福印
3. 5.4–7.8s｜翻牌 → "Phát tài" + 金山宝箱图 → **发财 fā cái**+朗读 → 词炸成金币雨落入福印（元时刻候选：phát tài 就是汉越词）
4. 7.8–10.2s｜翻牌 → "Tình yêu đích thực" + 并生樱花树图 → **真爱 zhēn ài**+朗读 → 红线从笔画抽出缠结、收入福印
5. 10.2–12.6s｜翻牌 → "Quý nhân" + 雾中灯塔图 → **贵人 guì rén**+朗读 → 金门开启光柱送词入印（元时刻候选）
6. 12.6–14.5s｜福印四格全亮 + "解"印封存 + "Gõ 'nhận' để nhận hết（打"接"全部收下）" + 预告："Ngày mai: những thứ sẽ tránh xa bạn.（明天：将远离你的东西。）"

## 与驱逐系的排播关系

隔日交替（今天驱逐、明天接收），预告互为对方引流——情绪节律"先解气后许愿"，账号像一档每日仪式栏目（P8 回访预期）。

## 本系专属红线

- 金钱词只送祝愿，永不承诺收益、不挂任何理财/带货暗示。
- "元时刻"每批视频最多点破 1–2 次（P7），过频会破坏仪式沉浸感。
