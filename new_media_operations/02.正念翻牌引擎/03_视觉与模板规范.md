# 视觉与模板规范（决策人 2026-07-12 拍板项，模板开发依据）

## 一、画幅与底色（拍板项，不可改）

- **画幅：1080 × 1440（3:4 竖屏）**，30fps。
- **背景色：与 TikTok For You 信息流底色一致的纯黑 `#000000`**。3:4 视频在 TikTok 9:16 播放器里上下留边，留边区域是 App 的黑底——视频底色与它同色，整条视频与信息流**融为一体、无边框感**。
- 上线前校准动作：真机截图 For You 页取色验证（个别机型/暗色模式下若实测偏离纯黑，以真机取色为准，全模板改一处变量即可）。
- 全片禁止任何贴近画幅边缘的亮色元素（会暴露画幅边界，破坏融入感）。

## 二、版面结构（2026-07-12 决策人终稿）

### 开场版面（≈3 秒）
1. **主题总结大图**满幅打底（本集词汇主题的合成意象，四边向黑渐隐）——必须震撼吸引有趣，全片第一冲击点；
2. **品牌四字"正念吸引"**：Ma Shan Zheng 毛笔体，大号，+ 越南语小注 `Chánh niệm · Luật hấp dẫn`（弱金色小字）；
3. **钩子句三层排版**：拼音（小，金）→ 中文（中，白/金）→ 越南语（大，粗体）自上而下堆叠；**只有越南语发音**，配音念完即切入词汇段；
4. 朱砂印章"解"右上角入场。

### 词汇版面（每词 2.2–2.5 秒，元素按时序入场）
1. **第一时序（0–0.9s）**：**越南语词**（Nunito 900 大号粗体，叠于图下沿）+ **该词动漫场景图**（占画面中部，四边渐隐）同时登场；
2. **第二时序（0.9–1.7s）**：**拼音 + 中文毛笔大字**砸入**图上方**——Ma Shan Zheng，2 字词 ≈ 340–380px（拍板项：字要非常大），驱逐系冷白（#F5F1E6）、接收系鎏金（#E9BE6A），拼音小字随词下；越南语词**不消失**；
3. **第三时序（1.7–2.4s）**：动效爆发——驱逐系词+图+背景全部毁灭清空；接收系词+图化金光收进**屏幕下方福印**（福印随收词逐格点亮）；
4. **常驻元素**：右上"解"印 + 缩小版"正念吸引"角标（与印同区）。
5. **底部**：片尾评论钩文案区（避开 TikTok 文案遮挡区）。

## 三、配图画风（拍板项）

- **夸张日漫画风：航海王 / 鬼灭之刃系**——粗线条、高对比、强动势线、戏剧性构图、饱和撞色。
- **无人物、无固定角色**：每个词一张纯场景/意象图，按词汇设计（渣男=暴雨中碎裂的玫瑰、发财=金山宝箱炸开、贵人=雾中灯塔）。
- **开场主题图**（每集一张，规格最高，**逐集全新设计、禁止复用**）：把本集 3–4 个词的意象合成为一个史诗级场景，怎么合成每集重新想，构图冲击力按"电影海报"标准验收——它承担首帧留人（P3），是全片最贵的一张图。
- 出图管线：fal flux + **动漫风格锚图**。首次搭建时先生成一张风格锚（demon-slayer/one-piece 风格的样例场景），人工确认画风后锁定，之后所有词图统一喂它（只借画风）。Prompt 骨架：
  `exaggerated Japanese anime style, bold ink outlines, dramatic composition, dynamic action lines, high contrast vivid colors, [该词的场景意象], no people, no characters, no text, dark background edges fading to black`
- 图幅 1:1 出图，版面内做上下裁切与边缘渐隐。

## 五、声音设计

- **双音色制（P10）**：越南语框架句 = vi_female_linh_uranus_bigtts（已实测）；**中文词朗读 = zh_female_kailangjiejie_moon_bigtts**（中文母语音色，读词干净）。音色档案：market/public/library/audio/voices/README.md。
- 每 2 秒一个音效打点（碎裂/撕纸/金币/钟磬），音效与动画同帧；片尾盖章声。
- BGM：低存在感的氛围底（禁舒缓古筝——实测催眠），驱逐系偏紧张脉冲、接收系偏上行音阶。

## 六、动效原语库（Remotion/React；动效逐词定制，原语只是积木）

**纪律：动效跟词走，一词一效**（决策人 2026-07-12 拍板）。下列原语是参数化积木组件（只增不删），每个词的最终动效 = 原语组合 + 专属参数 + 专属音效，写在该词 script.json 的 fx 字段里，逐词验收：

- 碎裂/撕裂：clip-path 多边形碎片 + 各碎片独立 spring 位移/旋转/坠落；
- 烧灼/溶解/蒸发：噪声遮罩阈值动画（feTurbulence/noise 纹理）+ 边缘描边色可调；
- 翻牌（接收系骨架动作）：3D rotateY + 高光扫过 + 鎏金粒子；
- 粒子系统：金币/花瓣/流光/灰烬/雨滴，发射器与落点可配（接收系落点统一屏幕下方中央——"落进你口袋"）；
- 光效：光柱/光晕荡开/脉冲发光/门缝透光；
- 描线生长：SVG stroke-dash 让红线/绿芽/桃枝"长"出来。
- 每 1.5 秒画面必须有变化（P5）——静帧超过 45 帧即违规。
