# 剧线 05 · 职场复仇《Trợ lý bị cướp công》（被抢功的助理）

**公式与证据**：复仇打脸——越南观众第二偏好题材（仅次于情感），单元剧结构。第 3 批试点（生产成本全引擎最低，如需垫量可提前启用，不受批次锁死）。

## 人物设定

- **An**（女主，26）：外贸公司小助理，方案常年被上司署名，忍功一流——夜里自学中文三年（App 特写位）。
- **Sếp Long**（反派上司，38）：抢功惯犯，靠 PPT 和人脉上位，中文水平只会"你好"。
- **Giám đốc Trang**（女总监，45）：明察秋毫但不动声色，An 的隐藏贵人。

**金手指用法**：中文 = 公开处刑的刑具。反派抢的是"纸面功劳"，而中文能力**当场可验、无法冒领**——中国客户一开口，谁真谁假立刻裸奔。每集打脸点全部设计在"语言当场验真"上，这是本线公式与金手指咬合最紧的地方。

## 结构说明：单元剧

与其他线不同，本线**每集一个完整打脸闭环**（冲突→冒领→当场验真→打脸），弱连续剧情、强模板复用——生产最省、试错最快，适合做引擎的"节奏担当"。集间只保留一条慢线：总监对 An 的观察升级。

## 5 集试点大纲（每集独立闭环）

- **EP1**：An 通宵写的对华报价方案被 Long 全文冒领；汇报会上中国客户视频连线突然提问，Long 卡壳冒汗，An 在桌下递纸条却被一把推开——客户直接点名："刚才门口用中文跟我打招呼的那位小姐呢？"（首集立机制）
- **EP2**：Long 把客户宴请安排成"自己主场"，点菜环节全靠装；客户用中文说了句玩笑话，全桌只有 An 笑了。客户："只有她听懂了我。"（微打脸 + 总监入场观察）
- **EP3**：Long 偷改 An 的翻译稿邀功，改错一个数字单位；签约现场 An 用一句中文当场纠正，避免公司损失，总监记下一笔。（本集中文台词位）
- **EP4**：Long 反咬 An "越级邀功"，散布谣言；中国客户来访点名要 An 全程陪同，Long 挤进合影却被客户助理礼貌请出画面。（社死打脸）
- **EP5**：年度述职，Long 的 PPT 与 An 的原始文档时间戳当场对比；总监宣布新设"中国区专员"，全场看向 An。Long 中文水平被要求当场展示——"你好"之后是漫长的沉默。（季终大打脸 + 升职钩）

## 本线专属规范

- 打脸必须"当场、公开、由第三方执行"（客户/总监出手，An 永远体面）——女主亲自撕反而掉爽感。
- 反派不降智：Long 的每次冒领手法要真实可信，打脸才有重量。
- 单元剧模板锁定后，LLM 批量生成"冒领手法 × 验真场景"组合，人工只筛打脸创意。
