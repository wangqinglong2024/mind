专病动态表单
王青龙
专病 · 动态表单系统 原型
管理端配置表单与类目，应用端按类目 Tab 采集数据。两端通过本地存储互通——在管理端保存/新增填报后，另一端刷新即可看到。
🛠️

管理端（配置）

动态表单配置器：页/卡片/字段增删改、宽度、是否入列表、条件属性；类目管理；一键预览（可切换移动端效果）。
📝

应用端（填报）

按专病/数据类目 Tab 展示动态列表（列由「是否在列表中显示」决定）；新增填报、查看记录；Web 与 H5 响应式自适应。
● 默认已内置含全部 8 种组件的示例表单（冠心病初诊表）及样例数据。
● 建议用 Chrome 打开；两端共享浏览器 localStorage 实现互通。
● 想恢复初始示例数据，点下方按钮。
