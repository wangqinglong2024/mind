# 组件配置：单选

## 组件说明
从若干选项中选择**一个**。用于性别、证型、疗效评级等。选项由医院配置时自行填写。

---

## 一、基础配置
| 配置项 | 说明 | 默认 |
|---|---|---|
| 字段名（标签） | 显示在字段前的名称 | 必填 |
| 是否必填 | 提交时必须选一项 | 否 |
| 提示说明 | 字段下方录入指引 | 无 |
| 默认选中 | 新建填报时预选的选项 | 无 |

## 二、选项配置
| 配置项 | 说明 | 默认 |
|---|---|---|
| 选项列表 | 逐条录入选项文本，支持**增、删、改、拖拽排序** | 必填≥2项 |
| 允许清空 | 已选后可取消选择回到空 | 是 |
| 其他项（补充输入） | 增加"其他"选项并允许填写文本 | 否 |

## 三、显示配置（含宽度）
> **宽度机制**：卡片内为栅格布局。字段设"栅格宽度"，同一行内宽度之和 ≤ 整行则并排，超出换行。**移动端(H5) 按《全局布局规则》降档：≤½ 变 ½（一行两列），>½ 变整行。**

| 配置项 | 可选值 | 默认 |
|---|---|---|
| 展示样式 | **固定为下拉选择**（统一下拉，避免选项过多撑坏宽度） | 下拉选择 |
| 栅格宽度 | 整行 / ½ / ⅓ / ⅔ / ¼ / ¾ | 整行 |
| 标签位置 | 顶部 / 左侧 | 顶部 |
| 标签宽度 | 标签在左侧时生效 | 自动 |
| 只读 | 只展示不可改 | 否 |
| 默认隐藏 | 初始隐藏，由条件逻辑控制 | 否 |
| **是否在列表中显示** | 应用端类目列表中是否作为一列展示（是/否） | 否 |

> 统一采用下拉选择：无论选项多少，宽度都稳定，不会因选项过多而撑坏布局。
