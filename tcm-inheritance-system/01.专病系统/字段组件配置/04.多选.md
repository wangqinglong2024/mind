# 组件配置：多选

## 组件说明
从若干选项中选择**多个**。用于症状、兼证、合并用药等。选项由医院配置时自行填写。

---

## 一、基础配置
| 配置项 | 说明 | 默认 |
|---|---|---|
| 字段名（标签） | 显示在字段前的名称 | 必填 |
| 是否必填 | 提交时至少选一项 | 否 |
| 提示说明 | 字段下方录入指引 | 无 |
| 默认选中 | 新建填报时预选的选项（可多个） | 无 |

## 二、选项配置
| 配置项 | 说明 | 默认 |
|---|---|---|
| 选项列表 | 逐条录入选项文本，支持**增、删、改、拖拽排序** | 必填≥2项 |
| 最少选择数 | 至少要选几项 | 不限 |
| 最多选择数 | 最多可选几项 | 不限 |
| 其他项（补充输入） | 增加"其他"选项并允许填写文本 | 否 |

## 三、显示配置（含宽度）
> **宽度机制**：卡片内为栅格布局。字段设"栅格宽度"，同一行内宽度之和 ≤ 整行则并排，超出换行。**移动端(H5) 按《全局布局规则》降档：≤½ 变 ½（一行两列），>½ 变整行。**

| 配置项 | 可选值 | 默认 |
|---|---|---|
| 展示样式 | **固定为多选下拉**（统一下拉，避免选项过多撑坏宽度） | 多选下拉 |
| 栅格宽度 | 整行 / ½ / ⅓ / ⅔ / ¼ / ¾ | 整行 |
| 标签位置 | 顶部 / 左侧 | 顶部 |
| 标签宽度 | 标签在左侧时生效 | 自动 |
| 只读 | 只展示不可改 | 否 |
| 默认隐藏 | 初始隐藏，由条件逻辑控制 | 否 |
| **是否在列表中显示** | 应用端类目列表中是否作为一列展示（是/否） | 否 |

> 统一采用多选下拉：选中项在框内以顿号连接展示，宽度稳定，不因选项多而撑坏布局。

## 四、高级配置
| 配置项 | 说明 |
|---|---|
| 条件显隐 | 可作逻辑触发源（判断"是否包含某项"）或目标 |
| 编辑 / 查看权限 | 指定可见、可改的角色 |
