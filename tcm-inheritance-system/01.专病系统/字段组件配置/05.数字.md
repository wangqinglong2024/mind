# 组件配置：数字

## 组件说明
数值输入。用于年龄、剂量、疗程天数、化验指标、随访次数等**可用于统计分析**的量。

---

## 一、基础配置
| 配置项 | 说明 | 默认 |
|---|---|---|
| 字段名（标签） | 显示在字段前的名称 | 必填 |
| 是否必填 | 提交时不能为空 | 否 |
| 占位提示 | 空值时框内灰字引导 | 无 |
| 提示说明 | 字段下方录入指引 | 无 |
| 默认值 | 新建填报时预填数值 | 无 |

## 二、校验配置
| 配置项 | 说明 | 默认 |
|---|---|---|
| 最小值 / 最大值 | 允许的取值范围（如年龄 0–120） | 不限 |
| 小数位数 | 保留几位小数（0 = 只允许整数） | 0 |
| 步长 | 加减按钮每次增减量 | 1 |
| 越界提示 | 超出范围时的提示文案 | 系统默认文案 |

## 三、显示配置（含宽度）
> **宽度机制**：卡片内为栅格布局。字段设"栅格宽度"，同一行内宽度之和 ≤ 整行则并排，超出换行。**移动端(H5) 按《全局布局规则》降档：≤½ 变 ½（一行两列），>½ 变整行。**
> 数字通常较短，**适合设 ⅓ / ¼ 与其他字段并排**（如"年龄""身高""体重"一行三列）。

| 配置项 | 可选值 | 默认 |
|---|---|---|
| 单位（后缀） | 显示在输入框右侧（如 g / 天 / 次 / mmHg） | 无 |
| 千分位显示 | 大数字以千分位分隔展示 | 否 |
| 栅格宽度 | 整行 / ½ / ⅓ / ⅔ / ¼ / ¾ | ½ |
| 标签位置 | 顶部 / 左侧 | 顶部 |
| 标签宽度 | 标签在左侧时生效 | 自动 |
| 只读 | 只展示不可改 | 否 |
| 默认隐藏 | 初始隐藏，由条件逻辑控制 | 否 |
| **是否在列表中显示** | 应用端类目列表中是否作为一列展示（是/否） | 否 |

## 四、高级配置
| 配置项 | 说明 |
|---|---|
| 条件显隐 | 可作逻辑触发源（如"某指标 > 阈值"显示提示）或目标 |
| 编辑 / 查看权限 | 指定可见、可改的角色 |
