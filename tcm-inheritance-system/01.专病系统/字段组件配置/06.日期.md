# 组件配置：日期

## 组件说明
日期 / 日期时间选择。用于就诊日期、初诊、复诊、随访日期、出生日期等。

---

## 一、基础配置
| 配置项 | 说明 | 默认 |
|---|---|---|
| 字段名（标签） | 显示在字段前的名称 | 必填 |
| 是否必填 | 提交时不能为空 | 否 |
| 提示说明 | 字段下方录入指引 | 无 |
| 默认值 | 无 / 当天 / 指定日期 | 无 |

## 二、格式与校验配置
| 配置项 | 说明 | 默认 |
|---|---|---|
| 日期精度 | 年月日 / 年月日时分 / 年月 / 仅年 | 年月日 |
| 可选范围 | 限制最早、最晚可选日期 | 不限 |
| 相对限制 | 如"不可晚于今天""不可早于就诊日期" | 无 |
| 越界提示 | 超范围时的提示文案 | 系统默认文案 |

## 三、显示配置（含宽度）
> **宽度机制**：卡片内为栅格布局。字段设"栅格宽度"，同一行内宽度之和 ≤ 整行则并排，超出换行。**移动端(H5) 按《全局布局规则》降档：≤½ 变 ½（一行两列），>½ 变整行。**
> 日期较短，**适合设 ½ / ⅓ 与其他字段并排**（如"初诊日期""复诊日期"一行两列）。

| 配置项 | 可选值 | 默认 |
|---|---|---|
| 栅格宽度 | 整行 / ½ / ⅓ / ⅔ / ¼ / ¾ | ½ |
| 标签位置 | 顶部 / 左侧 | 顶部 |
| 标签宽度 | 标签在左侧时生效 | 自动 |
| 只读 | 只展示不可改 | 否 |
| **是否在列表中显示** | 应用端类目列表中是否作为一列展示（是/否） | 否 |
