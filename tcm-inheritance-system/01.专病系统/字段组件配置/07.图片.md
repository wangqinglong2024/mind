# 组件配置：图片

## 组件说明
图片上传。用于舌象、面象、脉图、检查报告、处方影印件等。中医望诊尤为关键。

---

## 一、基础配置
| 配置项 | 说明 | 默认 |
|---|---|---|
| 字段名（标签） | 显示在字段前的名称 | 必填 |
| 是否必填 | 提交时至少上传一张 | 否 |
| 提示说明 | 字段下方录入指引（如"请上传伸舌自然光照片"） | 无 |

## 二、上传规则配置
| 配置项 | 说明 | 默认 |
|---|---|---|
| 数量上限 | 最多可传张数 | 9 |
| 单张大小上限 | 超过则拒绝或压缩（如 10MB） | 10MB |
| 允许格式 | jpg / png / heic 等 | jpg、png |
| 来源（H5） | 拍照 / 相册 / 两者 | 两者 |
| 自动压缩 | 上传前按上限压缩，减小体积 | 是 |

## 三、显示配置
> ⚠️ **本组件固定整行、不提供宽度设置**（图片为缩略图网格、块级，独占整行）。详见《全局布局规则》第五节。

| 配置项 | 可选值 | 默认 |
|---|---|---|
| 缩略图每行张数 | 3 / 4 / 5 | 4 |
| 栅格宽度 | 固定整行（不可配置） | 整行 |
| 标签位置 | 顶部 / 左侧 | 顶部 |
| 只读 | 只展示不可改（仅预览、不可增删） | 否 |
| 默认隐藏 | 初始隐藏，由条件逻辑控制 | 否 |
| **是否在列表中显示** | 应用端类目列表中是否作为一列展示（是/否，图片建议否或显示缩略图） | 否 |

## 四、高级配置
| 配置项 | 说明 |
|---|---|
| 条件显隐 | 按其他字段值控制显隐 |
| 查看权限 | 指定哪些角色可查看图片（隐私敏感） |
