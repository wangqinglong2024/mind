# 组件配置：子表（表格）

## 组件说明
表格容器：可配置**多列**，每列是前 7 种基础组件之一。应用端以表格呈现，用户可**加一行 / 删一行**，反复录入多条。典型用途：处方（多味药）、多次随访。

- 子表内**不再嵌套子表**（只允许前 7 种）。
- 加行时表格**向下延展、不分页**；行多时随整页滚动，**表头吸顶**保证列名不丢。

---

## 一、基础配置
| 配置项 | 说明 | 默认 |
|---|---|---|
| 字段名（标签） | 子表标题（如"处方用药"） | 必填 |
| 提示说明 | 表格上方录入指引 | 无 |
| 空态提示 | 一行都没有时的引导文案 | "点击下方添加" |

## 二、列配置（核心）
每个子表由若干**列**组成，逐列配置：

| 列配置项 | 说明 |
|---|---|
| 列标签 | 表头显示名（如 药名 / 剂量 / 单位 / 炮制） |
| 列类型 | 前 7 种之一：单行文本 / 多行文本 / 单选 / 多选 / 数字 / 日期 / 图片 |
| 列内组件配置 | 复用对应组件的配置（如数字列的最小值/单位、单选列的选项列表、文本列的格式校验等） |
| 是否必填 | 该列每行是否必填 |
| 列宽 | 该列在表格中占的宽度（比例或自适应） |
| 列顺序 | 拖拽调整列的左右顺序 |

> 例：处方子表 = 药名(单行文本) + 剂量(数字, 单位 g) + 单位(单选: g/ml/枚) + 炮制(单行文本)。

## 三、行规则配置
| 配置项 | 说明 | 默认 |
|---|---|---|
| 允许加行 | 用户可否新增行 | 是 |
| 最少行数 | 提交时至少要有几行 | 0 |
| 最多行数 | 最多可加几行 | 不限 |
| 允许删行 | 用户可否删除行 | 是 |
| 行拖拽排序 | 用户可否上下拖动调整行序 | 是 |
| 显示序号列 | 表格最左显示行号 | 是 |
| 默认行数 | 新建填报时预置的空白行数 | 1 |

## 四、显示配置
> ⚠️ **本组件固定整行、不提供宽度设置**（子表为表格、块级，独占整行，不与其他字段并排）。详见《全局布局规则》第五节。
> **移动端(H5)** 表格可横向滚动，或按行卡片化展示（每行折叠为一张小卡）。

| 配置项 | 可选值 | 默认 |
|---|---|---|
| 栅格宽度 | 固定整行（不可配置） | 整行 |
| 表头吸顶 | 滚动时列名固定在顶部 | 是 |
| 默认隐藏 | 初始隐藏，由条件逻辑控制 | 否 |
| H5 展示方式 | 横向滚动表格 / 行卡片化 | 行卡片化 |
| **是否在列表中显示** | 应用端类目列表中是否作为一列展示（是/否，子表建议否，或显示行数摘要） | 否 |

## 五、高级配置
| 配置项 | 说明 |
|---|---|
| 条件显隐 | 按其他字段值控制整个子表显隐 |
| 编辑 / 查看权限 | 指定可见、可改的角色 |
