# 需求 · 应用端 · 类目与数据列表

> 对应原型：应用端首页。浏览路径：**选类目 → 选模板 → 查看该模板的数据列表**。整体全屏宽度。

---

## 类目 Tab（第一步：选类目）
- 顶部以 Tab 展示**含有模板的类目**（空类目不出现）。
- 默认选中第一个类目，点击切换。
- Tab 可横向滚动，适配数量较多与小屏。

## 模板选择（第二步：选模板）
- 类目下方展示该类目的**模板选择条**（一个类目可有多张模板）。
- 默认选中第一张模板；点击切换模板。
- 切换类目时，模板选择重置为该类目的第一张。

## 数据列表（第三步：看记录）
- 展示当前**所选模板**的数据记录列表，标题为模板名称。
- **列动态生成**：数据列由该模板中**「是否在列表中显示」为是**的字段决定，顺序同字段顺序。
- 固定附加列：序号、提交时间、操作（查看）。
- 记录按提交时间倒序排列。
- 无记录时显示空态，引导点击「新增填报」。

## 列内容展示规则
- 文本 / 单选 / 日期：直接显示值。
- 多选 / 标签文本：多值以**顿号**连接展示。
- 数字：带单位时附加单位。
- 图片：显示"📷 N 张"摘要。
- 列表（子表）：显示"N 条"摘要。

## 操作入口
- **新增填报**：列表右上入口，进入整页填报（见《06》）。
- **查看**：记录行「查看」，进入整页只读查看（见《06》）。

## 响应式
- **Web**：以**表格**展示列表。
- **移动端（<768px）**：列表改为**卡片式**，每条记录一张卡片，卡片内以"字段名—值"逐行展示同样的列。
