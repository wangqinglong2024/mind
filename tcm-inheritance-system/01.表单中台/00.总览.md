# 表单中台 · 需求总览

> 原「专病动态表单」升级为**表单中台**：表单的**配置与渲染**中台共用，多个业务系统（问诊单、医案采集…）都可接入；"表单如何分类、归属于谁"是各业务自己的事，与中台**解耦**。
> 本套文档在同事已上线的「问诊单设置」基础上**补充完善**，已上线功能全部保留、不删除。
> 配套原型见 `../原型设计/`，字段组件细则见 `../字段组件配置/`。

---

## 一、分层（核心变更）

原来把"类目"和"表单配置"焊在一个页面、只考虑本系统；现在**彻底分层**：

1、**表单中台（共用层，与业务无关）**
   层级：`表单 → 分页 → 题目`
   负责：题型/组件、题目属性、选项、关联症状/疾病、查询条件、分页与分页主题、预览、渲染。

2、**业务侧（各系统自己的事，与中台解耦）**
   负责：分类/类目、表单的投放、适用范围（如问诊单的"投放状态""适用医生"）、数据列表。
   注：**问诊单只是中台的一个使用方**；换一个业务（医案采集）接同一套表单中台即可，分类是它自己的另一套东西。

3、拉平结构、砍复杂度：
   - 删除"卡片/分组"这一层，只保留 `分页 → 题目`，与已上线的问诊单一致。
   - 删除"1/4/⅓ 栅格宽度"整套，**所有题目独占整行**。
   - 总原则：**傻瓜化操作**，能少一步是一步，不做复杂交互。

## 二、题型 / 组件（整行布局，选择题非下拉）

| 分类 | 题型 | 一句话 |
|---|---|---|
| 选择 | **单选** | 单选框列表选一个（问卷网样式，**禁止下拉**） |
| 选择 | **多选** | 勾选框列表选多个（问卷网样式，**禁止下拉**） |
| 选择 | **单选打分** | 单选 + 每个选项可设分值 |
| 选择 | **多选打分** | 多选 + 每个选项可设分值 |
| 填空 | **单行文本 / 填空** | 单行输入 |
| 填空 | **多行文本** | 多行输入 |
| 填空 | **数字** | 数值输入，可设范围 |
| 填空 | **日期时间** | 日期 / 日期时间选择 |
| 展示 | **富文本** | 仅展示说明文字/图文，不采集（新增） |
| 采集 | **图片** | 图片上传（舌象、检查报告等） |
| 采集 | **标签文本** | 多个离散标签，可按标签统计 |
| 采集 | **列表 / 子表** | 表格容器，可配多列、应用端可加行（如处方多味药） |

注1：单选/多选/单选打分/多选打分/填空为同事已上线题型，全部保留。
注2：考虑未来拓展，题型可继续新增。
注3：单选/多选/单选打分/多选打分的**每个选项可关联 1 个症状或疾病**，后端记录关联关系便于数据分析（见《关联》）。

## 三、在同事版本上补充的能力

1、**查询条件（新增）**
   单行文本 / 数字 / 日期时间 / 选择（单选、多选）等题目，可在中台配置时勾选"设为查询条件"并选择**查询形式**：
   - 文本：模糊匹配 / 精准匹配
   - 数字：数字范围 / 精准匹配
   - 日期时间：日期时间范围 / 精准匹配
   - 选择：单选下拉 / 多选筛选
   （以上为部分举例，凡可查询的题型都提供该设置。）
   配置后：**应用端数据列表上方出现对应筛选条件**；字段及规则可**传给算法**做进一步计算。

2、**富文本组件（新增）**：在现有组件基础上增加，用于展示说明，不参与采集。

3、**搭建器改问卷网风格（新增/改造）**：
   - 三栏：**左（组件选择）/ 中（画布展示）/ 右（题目属性）**。
   - 题目支持**拖拽上下排序**、可直接删除（替代点击"上移/下移"）。
   - 属性面板样式参考问卷网重做。

4、**预览改问卷网风格**：新窗口，**手机 / 电脑切换**，移动端居中显示手机框。

## 四、两个端

| 端 | 使用者 | 职责 |
|---|---|---|
| **搭建端（中台）** | 配置人员 | 拖拽搭建表单：题型、题目、选项、关联、查询条件、分页、预览 |
| **应用端（业务）** | 录入员 / 医生 | 按业务分类查看数据列表（带查询条件）、按表单填报、查看记录 |
