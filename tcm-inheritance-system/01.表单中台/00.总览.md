# 表单中台 · 需求总览

> 原「专病动态表单」升级为**表单中台**：表单的**配置与渲染**中台共用，多个业务系统（问诊单、医案采集…）都可接入；"表单如何分类、归属于谁、是否发布"是各业务自己的事，与中台**解耦**。
> 配套原型见 `../原型设计/`，字段组件细则见 `字段组件配置/`。

---

## 一、分层

1、**表单中台（共用层，与业务无关）**
   层级：`表单 → 分页 → 卡片 → 题目`
   负责：题型/组件、题目属性、选项、关联字段（症状/疾病）、查询条件、分页与分页主题、卡片分组、预览、渲染。

2、**业务侧（各系统自己的事，与中台解耦）**
   负责：类目/分类、表单的**发布/下架**、数据列表与采集。
   注：**一个业务（如问诊单）只是中台的一个使用方**；换一个业务（医案采集）接同一套表单中台即可，类目是它自己的另一套东西。

3、结构与取舍：
   - 层级为 `分页 → 卡片 → 题目`：一页可含多个**卡片**（分组容器，卡片标题选填、填写者可见），卡片内放题目。
   - **不做 1/4/⅓ 栅格宽度**，所有题目**独占整行**。
   - 总原则：**傻瓜化操作**，能少一步是一步，不做复杂交互。

## 二、题型 / 组件（整行布局，选择题非下拉）

| 分组 | 题型 | 一句话 |
|---|---|---|
| 选择 | **单选** | 单选框列表选一个（问卷网样式，**禁止下拉**），可设默认选项 |
| 选择 | **多选** | 勾选框列表选多个（问卷网样式，**禁止下拉**） |
| 选择 | **单选打分** | 单选 + 每个选项可设分值，可设默认选项 |
| 选择 | **多选打分** | 多选 + 每个选项可设分值 |
| 填空 | **单行文本** | 单行输入 |
| 填空 | **多行文本** | 多行输入 |
| 填空 | **数字** | 数值输入，可设范围/单位 |
| 填空 | **日期时间** | 日期 / 日期时间选择 |
| 采集 | **图片** | 图片上传（舌象、检查报告等） |
| 采集 | **标签文本** | 多个离散标签，回车新增，可按标签统计 |
| 采集 | **列表** | 表格容器，可配多列、应用端可加行（如处方多味药） |
| 采集 | **富文本** | **可填的富文本字段**：填写者录入格式化内容，查看时按富文本渲染 |

注1：单选/多选/单选打分/多选打分/填空为已上线题型，全部保留。
注2：考虑未来拓展，题型可继续新增。
注3：选择题（单选/多选/单选打分/多选打分）可开启"**选项关联字段**"开关，开启后每个选项可**关联 1 个症状或疾病**，后端记录关联关系便于数据分析。

## 三、在同事版本上补充/调整的能力

1、**卡片分组**：分页内可分多个卡片，卡片标题选填、填写者可见；题目可在卡片间拖拽。

2、**查询条件**：可查询题型在中台配置时勾选"设为查询条件"并选择**查询形式**：
   | 题型 | 查询形式 |
   |---|---|
   | 单选 / 多选 / 单选打分 / 多选打分 | 单选查询 / 多选查询 |
   | 单行文本 / 多行文本 / 富文本 / 标签文本 / 列表 | 模糊查询 / 精准查询 |
   | 数字 | 数字范围 |
   | 日期时间 | 日期时间范围 |
   | 图片 | 不支持查询 |
   配置后：应用端数据列表以**右侧抽屉**呈现对应筛选；字段及规则可**传给算法**。

3、**在列表中显示**：可作为数据列表列的题型仅 **单选 / 单选打分 / 单行文本 / 数字 / 日期**。

4、**富文本改为可填字段**：不再是"仅展示"块。搭建时只在属性面板配"占位提示"；预览/填报是富文本编辑器；查看数据按富文本渲染；可作查询条件（模糊/精准）。

5、**搭建器问卷网风格**：三栏（左组件 / 中画布 / 右属性）；题目**拖拽排序**、直接删除；**题目标题在中间画布内直接编辑**（不在属性面板）；卡片底部"＋ 添加题目"点击弹出分组题型卡片。

6、**预览问卷网风格**：新窗口、**手机/电脑切换**、可交互填报、多页翻页、顶部【返回】。

## 四、两个端

| 端 | 使用者 | 职责 |
|---|---|---|
| **搭建端（中台）** | 配置人员 | 类目 + 表单列表(发布/下架/隐藏/删除) + 拖拽搭建器（题型、题目、选项、关联、查询条件、卡片、分页、预览） |
| **应用端（业务）** | 录入员 / 医生 | 顶部二级导航选表单 → 数据列表（查询抽屉）→ 查看/编辑/删除/新增采集 |
