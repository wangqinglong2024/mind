# 组件配置：单行文本

## 组件说明
单行文本输入框。用于短文本，如患者姓名、方名、诊断名称、编号等。

---

## 一、基础配置（题目标题在画布内编辑）
| 配置项 | 说明 | 默认 |
|---|---|---|
| 题目标题 | 在画布题目行内直接编辑，长度 [1,20] | 必填 |
| 是否必填 | 提交时不能为空 | 否 |
| 占位提示 | 空值时输入框内的灰字引导 | 无 |
| 题目说明 | 标题下方的录入指引 | 无 |
| 默认值 | 新建填报时预填的内容 | 无 |

## 二、校验配置
| 配置项 | 说明 | 默认 |
|---|---|---|
| 最小长度 / 最大长度 | 限制字符数 | 不限 |
| 格式校验 | 无 / 手机号 / 身份证 / 邮箱 / 纯数字 | 无 |

## 三、显示配置
> 布局**固定整行**，无栅格宽度。

| 配置项 | 可选值 | 默认 |
|---|---|---|
| 布局 | 固定整行 | 整行 |
| 在列表中显示 | 应用端数据列表是否作为一列（**支持**） | 否 |

## 四、查询条件
| 配置项 | 说明 | 默认 |
|---|---|---|
| 设为查询条件 | 应用端查询抽屉出现该字段筛选 | 关 |
| 查询形式 | 模糊查询 / 精准查询 | 模糊查询 |

注：字段标识+查询规则可传给算法。
