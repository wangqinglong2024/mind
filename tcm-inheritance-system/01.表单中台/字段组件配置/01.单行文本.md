# 组件配置：单行文本

## 组件说明
单行文本输入框。用于短文本，如患者姓名、方名、诊断名称、编号等。

---

## 一、基础配置
| 配置项 | 说明 | 默认 |
|---|---|---|
| 字段名（标签） | 显示在字段前的名称 | 必填 |
| 是否必填 | 提交时不能为空 | 否 |
| 占位提示 | 空值时输入框内的灰字引导 | 无 |
| 提示说明 | 字段下方的录入指引 | 无 |
| 默认值 | 新建填报时预填的内容 | 无 |

## 二、校验配置
| 配置项 | 说明 | 默认 |
|---|---|---|
| 最小长度 / 最大长度 | 限制字符数 | 不限 |
| 格式校验 | 无 / 手机号 / 身份证 / 邮箱 / 纯数字 / 自定义正则 | 无 |
| 格式错误提示 | 校验不通过时的提示文案 | 系统默认文案 |
| 前缀 / 后缀 | 输入框左右的固定文字（如单位、编号前缀） | 无 |

## 三、显示配置（含宽度）
> **宽度机制**：卡片内为栅格布局。每个字段设"栅格宽度"，同一行内字段宽度之和 ≤ 整行则自动并排，超出则换到下一行。例：三个字段都设 ⅓ → 一行三列。**移动端(H5) 按《全局布局规则》降档：≤½ 变 ½（一行两列），>½ 变整行。**

| 配置项 | 可选值 | 默认 |
|---|---|---|
| 栅格宽度 | 整行 / ½ / ⅓ / ⅔ / ¼ / ¾ | 整行 |
| 标签位置 | 顶部 / 左侧 | 顶部 |
| 标签宽度 | 标签在左侧时生效（如 4em / 6em） | 自动 |
| 只读 | 只展示不可改 | 否 |
| **是否在列表中显示** | 应用端类目列表中是否作为一列展示（是/否） | 否 |
