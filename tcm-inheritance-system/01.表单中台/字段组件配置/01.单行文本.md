# 组件配置：单行文本

## 组件说明
单行文本输入框。用于短文本，如患者姓名、方名、诊断名称、编号等。

---

## 一、基础配置
| 配置项 | 说明 | 默认 |
|---|---|---|
| 字段名（标签） | 显示在字段前的名称 | 必填 |
| 是否必填 | 提交时不能为空 | 否 |
| 占位提示 | 空值时输入框内的灰字引导 | 无 |
| 提示说明 | 字段下方的录入指引 | 无 |
| 默认值 | 新建填报时预填的内容 | 无 |

## 二、校验配置
| 配置项 | 说明 | 默认 |
|---|---|---|
| 最小长度 / 最大长度 | 限制字符数 | 不限 |
| 格式校验 | 无 / 手机号 / 身份证 / 邮箱 / 纯数字 / 自定义正则 | 无 |
| 格式错误提示 | 校验不通过时的提示文案 | 系统默认文案 |
| 前缀 / 后缀 | 输入框左右的固定文字（如单位、编号前缀） | 无 |

## 三、显示配置
> 布局**固定整行**，无栅格宽度（旧栅格机制已删除）。

| 配置项 | 可选值 | 默认 |
|---|---|---|
| 布局 | 固定整行 | 整行 |
| 只读 | 只展示不可改 | 否 |
| 在列表中显示为列 | 应用端数据列表是否作为一列（是/否） | 否 |

## 四、查询条件
| 配置项 | 说明 | 默认 |
|---|---|---|
| 设为查询条件 | 应用端列表上方出现该字段筛选 | 关 |
| 查询形式 | 模糊匹配 / 精准匹配 | 模糊匹配 |

注：字段标识+查询规则可传给算法。
