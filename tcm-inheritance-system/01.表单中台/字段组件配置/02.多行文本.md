# 组件配置：多行文本

## 组件说明
多行文本输入框。用于较长的叙述性文本，如主诉、现病史、刻下症、按语等。

---

## 一、基础配置
| 配置项 | 说明 | 默认 |
|---|---|---|
| 字段名（标签） | 显示在字段前的名称 | 必填 |
| 是否必填 | 提交时不能为空 | 否 |
| 占位提示 | 空值时框内灰字引导 | 无 |
| 提示说明 | 字段下方录入指引 | 无 |
| 默认值 | 新建填报时预填内容 | 无 |

## 二、校验配置
| 配置项 | 说明 | 默认 |
|---|---|---|
| 最大字数 | 限制总字符数 | 不限 |
| 显示字数统计 | 输入框右下角显示 `已输入/上限` | 否 |
| 默认显示行数 | 输入框初始高度（行） | 3 行 |
| 是否可拉伸 | 允许用户拖动改变高度 | 是 |

## 三、显示配置
> 块级组件，**固定整行**，无栅格宽度。详见《需求文档/04.整行布局规则》。

| 配置项 | 可选值 | 默认 |
|---|---|---|
| 布局 | 固定整行 | 整行 |
| 只读 | 只展示不可改 | 否 |
| 在列表中显示为列 | 是/否（长文本建议否） | 否 |
