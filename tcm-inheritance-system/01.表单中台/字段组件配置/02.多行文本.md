# 组件配置：多行文本

## 组件说明
多行文本输入框。用于较长的叙述性文本，如主诉、现病史、刻下症、按语等。

---

## 一、基础配置
| 配置项 | 说明 | 默认 |
|---|---|---|
| 题目标题 | 在画布题目行内直接编辑，[1,20] | 必填 |
| 是否必填 | 提交时不能为空 | 否 |
| 占位提示 | 空值时框内灰字引导 | 无 |
| 题目说明 | 标题下方录入指引 | 无 |
| 默认值 | 新建填报时预填内容 | 无 |

## 二、校验配置
| 配置项 | 说明 | 默认 |
|---|---|---|
| 最大字数 | 限制总字符数 | 不限 |
| 默认显示行数 | 输入框初始高度（行） | 3 行 |

## 三、显示配置
> 块级组件，**固定整行**，无栅格宽度。

| 配置项 | 可选值 | 默认 |
|---|---|---|
| 布局 | 固定整行 | 整行 |
| 在列表中显示 | **不支持**（长文本不作列） | — |

## 四、查询条件
| 配置项 | 说明 | 默认 |
|---|---|---|
| 设为查询条件 | 应用端查询抽屉出现该字段筛选 | 关 |
| 查询形式 | 模糊查询 / 精准查询 | 模糊查询 |
