# 组件配置：单选

## 组件说明
从若干选项中选择**一个**。用于性别、证型、疗效评级等。选项由配置者填写，或**关联症状/疾病**。
样式：**问卷网单选框竖向列表（禁止下拉）**，独占整行。

---

## 一、基础配置
| 配置项 | 说明 | 默认 |
|---|---|---|
| 题目标题 | 画布内编辑，[1,20] | 必填 |
| 是否必填 | 提交时必须选一项 | 否 |
| 题目说明 | 标题下方录入指引 | 无 |
| 允许清空 | 已选后可取消选择回到空 | 是 |

## 二、默认选项
| 配置项 | 说明 | 默认 |
|---|---|---|
| 设置默认选项 | 题目底部开关；**开启后**可点选 1 项为默认选中，关闭则不预选、不提示 | 关 |

## 三、选项配置
| 配置项 | 说明 | 默认 |
|---|---|---|
| 选项列表 | 逐条录入，支持增、删、改 | 新加默认 3 项 |
| 选项文字 | 输入文字 [0,20] | 空 |
| 选项关联字段 | 题目底部开关（默认关）；**开启后**每个选项出现【↔关联】，可关联 1 个症状/疾病，成功后隐藏文字框、展示关联项+【x】 | 关 |

关联规则见《需求文档/02·六》。关联项后端记录 id 供分析。

## 四、显示配置
| 配置项 | 可选值 | 默认 |
|---|---|---|
| 展示样式 | 单选框竖向列表（禁止下拉） | 列表 |
| 选项排列 | **单列 / 双列**（属性面板设置，画布联动，控制应用端 1/2 列） | 单列 |
| 布局 | 固定整行 | 整行 |
| 在列表中显示 | 应用端数据列表作为一列（**支持**） | 否 |

## 五、查询条件
| 配置项 | 说明 | 默认 |
|---|---|---|
| 设为查询条件 | 应用端查询抽屉出现筛选 | 关 |
| 查询形式 | **单选查询 / 多选查询**（均为下拉） | 单选查询 |
