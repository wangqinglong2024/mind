# 组件配置：单选

## 组件说明
从若干选项中选择**一个**。用于性别、证型、疗效评级等。选项由配置者自行填写，或**关联症状/疾病**。
样式：**问卷网单选框竖向列表（禁止下拉）**，独占整行。

---

## 一、基础配置
| 配置项 | 说明 | 默认 |
|---|---|---|
| 题目标题 | 显示在题目前的名称，长度 [1,20] | 必填 |
| 是否必填 | 提交时必须选一项 | 否 |
| 题目说明 | 标题下方录入指引 | 无 |
| 默认选中 | 新建填报时预选的选项 | 无 |

## 二、选项配置
| 配置项 | 说明 | 默认 |
|---|---|---|
| 选项列表 | 逐条录入，支持增、删、改、拖拽排序 | 新加默认 3 项 |
| 选项内容 | 输入文字（[0,20]）**或**关联 1 个症状/疾病，二选一 | 空 |
| 关联 | 点【关联】选症状/疾病，成功后隐藏文字框，展示关联项+【x】 | — |
| 允许清空 | 已选后可取消选择回到空 | 是 |

关联规则详见《需求文档/02.题目属性与查询条件·五》。关联项后端记录 id 供分析。

## 三、显示配置
| 配置项 | 可选值 | 默认 |
|---|---|---|
| 展示样式 | **单选框竖向列表**（问卷网样式，禁止下拉） | 列表 |
| 布局 | **固定整行**（无栅格宽度） | 整行 |
| 只读 | 只展示不可改 | 否 |
| 在列表中显示为列 | 应用端数据列表是否作为一列 | 否 |

## 四、查询条件
| 配置项 | 说明 | 默认 |
|---|---|---|
| 设为查询条件 | 开启后，应用端列表上方出现该字段筛选 | 关 |
| 查询形式 | 单选下拉筛选 | 单选下拉 |

注：字段标识+查询规则可传给算法。
