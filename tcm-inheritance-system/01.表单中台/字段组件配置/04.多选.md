# 组件配置：多选

## 组件说明
从若干选项中选择**多个**。用于症状、兼证、合并用药等。选项由配置者自行填写，或**关联症状/疾病**。
样式：**问卷网勾选框竖向列表（禁止下拉）**，独占整行。

---

## 一、基础配置
| 配置项 | 说明 | 默认 |
|---|---|---|
| 题目标题 | 长度 [1,20] | 必填 |
| 是否必填 | 提交时至少选一项 | 否 |
| 题目说明 | 标题下方录入指引 | 无 |
| 默认选中 | 新建填报时预选（可多个） | 无 |

## 二、选项配置
| 配置项 | 说明 | 默认 |
|---|---|---|
| 选项列表 | 增、删、改、拖拽排序 | 新加默认 3 项 |
| 选项内容 | 输入文字（[0,20]）**或**关联 1 个症状/疾病 | 空 |
| 关联 | 同单选，成功后隐藏文字框展示关联项+【x】 | — |
| 最少 / 最多选择数 | 限制勾选数量 | 不限 |

## 三、显示配置
| 配置项 | 可选值 | 默认 |
|---|---|---|
| 展示样式 | **勾选框竖向列表**（禁止下拉） | 列表 |
| 布局 | **固定整行** | 整行 |
| 只读 | 只展示不可改 | 否 |
| 在列表中显示为列 | 是/否 | 否 |

## 四、查询条件
| 配置项 | 说明 | 默认 |
|---|---|---|
| 设为查询条件 | 应用端列表上方出现多选筛选 | 关 |
| 查询形式 | 多选筛选 | 多选 |
