# 组件配置：多选

## 组件说明
从若干选项中选择**多个**。用于症状、兼证、合并用药等。选项由配置者填写，或**关联症状/疾病**。
样式：**问卷网勾选框竖向列表（禁止下拉）**，独占整行。

---

## 一、基础配置
| 配置项 | 说明 | 默认 |
|---|---|---|
| 题目标题 | 画布内编辑，[1,20] | 必填 |
| 是否必填 | 提交时至少选一项 | 否 |
| 题目说明 | 标题下方录入指引 | 无 |
| 最少 / 最多选择数 | 限制勾选数量 | 不限 |

注：多选**不提供**"默认选项"（仅单选/单选打分有）。

## 二、选项配置
| 配置项 | 说明 | 默认 |
|---|---|---|
| 选项列表 | 增、删、改 | 新加默认 3 项 |
| 选项文字 | 输入文字 [0,20] | 空 |
| 选项关联字段 | 题目底部开关（默认关）；开启后每个选项可【↔关联】1 个症状/疾病 | 关 |

## 三、显示配置
| 配置项 | 可选值 | 默认 |
|---|---|---|
| 展示样式 | 勾选框竖向列表（禁止下拉） | 列表 |
| 选项排列 | **单列 / 双列**（属性面板设置，画布联动） | 单列 |
| 布局 | 固定整行 | 整行 |
| 在列表中显示 | **不支持** | — |

## 四、查询条件
| 配置项 | 说明 | 默认 |
|---|---|---|
| 设为查询条件 | 应用端查询抽屉出现筛选 | 关 |
| 查询形式 | **单选查询 / 多选查询**（均为下拉） | 单选查询 |
