# 组件配置：数字

## 组件说明
数值输入。用于年龄、剂量、疗程天数、化验指标、随访次数等**可用于统计分析**的量。

---

## 一、基础配置
| 配置项 | 说明 | 默认 |
|---|---|---|
| 题目标题 | 画布内编辑，[1,20] | 必填 |
| 是否必填 | 提交时不能为空 | 否 |
| 占位提示 | 空值时框内灰字引导 | 无 |
| 题目说明 | 标题下方录入指引 | 无 |
| 默认值 | 新建填报时预填数值 | 无 |

## 二、校验配置
| 配置项 | 说明 | 默认 |
|---|---|---|
| 最小值 / 最大值 | 允许的取值范围（如年龄 0–120） | 不限 |
| 小数位数 | 保留几位小数（0 = 只允许整数） | 0 |

## 三、显示配置
> 布局**固定整行**，无栅格宽度。

| 配置项 | 可选值 | 默认 |
|---|---|---|
| 单位（后缀） | 显示在输入框右侧（如 g / 天 / 次 / ℃） | 无 |
| 布局 | 固定整行 | 整行 |
| 在列表中显示 | **支持** | 否 |

## 四、查询条件
| 配置项 | 说明 | 默认 |
|---|---|---|
| 设为查询条件 | 应用端查询抽屉出现该字段筛选 | 关 |
| 查询形式 | **数字范围**（最小值 ~ 最大值） | 数字范围 |
