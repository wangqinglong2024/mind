# 组件配置：数字

## 组件说明
数值输入。用于年龄、剂量、疗程天数、化验指标、随访次数等**可用于统计分析**的量。

---

## 一、基础配置
| 配置项 | 说明 | 默认 |
|---|---|---|
| 字段名（标签） | 显示在字段前的名称 | 必填 |
| 是否必填 | 提交时不能为空 | 否 |
| 占位提示 | 空值时框内灰字引导 | 无 |
| 提示说明 | 字段下方录入指引 | 无 |
| 默认值 | 新建填报时预填数值 | 无 |

## 二、校验配置
| 配置项 | 说明 | 默认 |
|---|---|---|
| 最小值 / 最大值 | 允许的取值范围（如年龄 0–120） | 不限 |
| 小数位数 | 保留几位小数（0 = 只允许整数） | 0 |
| 步长 | 加减按钮每次增减量 | 1 |
| 越界提示 | 超出范围时的提示文案 | 系统默认文案 |

## 三、显示配置
> 布局**固定整行**，无栅格宽度。

| 配置项 | 可选值 | 默认 |
|---|---|---|
| 单位（后缀） | 显示在输入框右侧（如 g / 天 / 次 / mmHg） | 无 |
| 千分位显示 | 大数字以千分位分隔展示 | 否 |
| 布局 | 固定整行 | 整行 |
| 只读 | 只展示不可改 | 否 |
| 在列表中显示为列 | 是/否 | 否 |

## 四、查询条件
| 配置项 | 说明 | 默认 |
|---|---|---|
| 设为查询条件 | 应用端列表上方出现该字段筛选 | 关 |
| 查询形式 | 数字范围 / 精准匹配 | 数字范围 |
