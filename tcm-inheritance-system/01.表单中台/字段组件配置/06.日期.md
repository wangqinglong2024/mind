# 组件配置：日期

## 组件说明
日期 / 日期时间选择。用于就诊日期、初诊、复诊、随访日期、出生日期等。

---

## 一、基础配置
| 配置项 | 说明 | 默认 |
|---|---|---|
| 字段名（标签） | 显示在字段前的名称 | 必填 |
| 是否必填 | 提交时不能为空 | 否 |
| 提示说明 | 字段下方录入指引 | 无 |
| 默认值 | 无 / 当天 / 指定日期 | 无 |

## 二、格式与校验配置
| 配置项 | 说明 | 默认 |
|---|---|---|
| 日期精度 | 年月日 / 年月日时分 / 年月 / 仅年 | 年月日 |
| 可选范围 | 限制最早、最晚可选日期 | 不限 |
| 相对限制 | 如"不可晚于今天""不可早于就诊日期" | 无 |
| 越界提示 | 超范围时的提示文案 | 系统默认文案 |

## 三、显示配置
> 布局**固定整行**，无栅格宽度。

| 配置项 | 可选值 | 默认 |
|---|---|---|
| 布局 | 固定整行 | 整行 |
| 只读 | 只展示不可改 | 否 |
| 在列表中显示为列 | 是/否 | 否 |

## 四、查询条件
| 配置项 | 说明 | 默认 |
|---|---|---|
| 设为查询条件 | 应用端列表上方出现该字段筛选 | 关 |
| 查询形式 | 日期时间范围 / 精准匹配 | 日期时间范围 |
