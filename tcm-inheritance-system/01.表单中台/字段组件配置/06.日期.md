# 组件配置：日期时间

## 组件说明
日期 / 日期时间选择。用于就诊日期、初诊、复诊、随访日期、出生日期等。

---

## 一、基础配置
| 配置项 | 说明 | 默认 |
|---|---|---|
| 题目标题 | 画布内编辑，[1,20] | 必填 |
| 是否必填 | 提交时不能为空 | 否 |
| 题目说明 | 标题下方录入指引 | 无 |
| 默认当天 | 新建填报时默认填当天 | 否 |

## 二、格式配置
| 配置项 | 说明 | 默认 |
|---|---|---|
| 日期精度 | 年月日 / 年月日时分 | 年月日 |

## 三、显示配置
> 布局**固定整行**，无栅格宽度。

| 配置项 | 可选值 | 默认 |
|---|---|---|
| 布局 | 固定整行 | 整行 |
| 在列表中显示 | **支持** | 否 |

## 四、查询条件
| 配置项 | 说明 | 默认 |
|---|---|---|
| 设为查询条件 | 应用端查询抽屉出现该字段筛选 | 关 |
| 查询形式 | **日期时间范围**（起 ~ 止） | 日期时间范围 |
