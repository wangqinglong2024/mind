# 组件配置：标签文本

## 组件说明
由**一个或多个标签**组成的字段。每个标签是一个独立的词，录入后可**基于标签进行统计**。可作为单行/多行文本的替代方案：既保留文本记录，又让内容结构化、可统计。

- **纯自由输入**：填报时手动输入词、回车即成一个标签。**不设预设候选词**（若需从固定选项里选，请改用「多选」）。
- 每个标签是离散值，天然适合后续做词频、分布等统计分析。

---

## 一、基础配置
| 配置项 | 说明 | 默认 |
|---|---|---|
| 字段名（标签） | 显示在字段前的名称 | 必填 |
| 是否必填 | 提交时至少一个标签 | 否 |
| 提示说明 | 字段下方录入指引 | 无 |
| 默认标签 | 新建填报时预置的标签 | 无 |

## 二、标签配置
| 配置项 | 说明 | 默认 |
|---|---|---|
| 最多标签数 | 单个字段最多可添加几个标签 | 不限 |
| 允许重复 | 同一字段内是否允许重复标签 | 否 |

## 三、显示配置
> 块级组件，**固定整行**，无栅格宽度；标签随录入数量动态换行增高。详见《需求文档/04.整行布局规则》。

| 配置项 | 可选值 | 默认 |
|---|---|---|
| 布局 | 固定整行 | 整行 |
| 只读 | 只展示不可改 | 否 |
| 在列表中显示为列 | 是/否（多标签以顿号连接） | 否 |
