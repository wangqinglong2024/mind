# 组件配置：列表

## 组件说明
表格容器：可配置**多列**，每列是一个基础组件。应用端以表格呈现，用户可**加一行 / 删一行**，反复录入多条。典型用途：处方（多味药）、多次随访。

- 列表内**不再嵌套列表**。
- 加行时表格**向下延展、不分页**；行多时随整页滚动，**表头吸顶**。

---

## 一、基础配置
| 配置项 | 说明 | 默认 |
|---|---|---|
| 题目标题 | 列表标题（如"用药记录"），画布内编辑 | 必填 |
| 题目说明 | 表格上方录入指引 | 无 |

## 二、列配置（核心）
在右栏属性面板逐列配置：

| 列配置项 | 说明 |
|---|---|
| 列名 | 表头显示名（如 药名 / 剂量 / 频次 / 用药方式 / 起始日期） |
| 列类型 | **单行文本 / 数字 / 日期 / 单选 / 多选**（5 种） |
| 列选项 | 当列类型为**单选 / 多选**时，配置其选项 |

> 应用端填报时：**单选 / 多选列以下拉选择**（多选为多选下拉）；单行文本/数字/日期为对应输入控件。
> 例：用药记录 = 药名(单行文本) + 剂量(单行文本) + 频次(单选：每日1次/2次/3次) + 用药方式(多选：口服/外用/注射) + 起始日期(日期)。

## 三、行规则
| 配置项 | 说明 | 默认 |
|---|---|---|
| 加行 / 删行 | 应用端可新增、删除行 | 允许 |

## 四、显示配置
> 块级组件，**固定整行**。移动端为**横向滚动表格**、表头吸顶。

| 配置项 | 可选值 | 默认 |
|---|---|---|
| 布局 | 固定整行 | 整行 |
| 在列表中显示 | **不支持**（数据列表可显示行数摘要，不作字段列） | — |

## 五、查询条件
| 配置项 | 说明 | 默认 |
|---|---|---|
| 设为查询条件 | 应用端查询抽屉出现该字段筛选 | 关 |
| 查询形式 | 模糊查询 / 精准查询 | 模糊查询 |
