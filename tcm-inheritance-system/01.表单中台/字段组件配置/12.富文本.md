# 组件配置：富文本

## 组件说明
**可填的富文本字段**：填写者在预览/填报时用富文本编辑器录入带格式的内容（加粗、颜色、列表等），查看数据时按**富文本渲染**呈现。
（区别于旧版"仅展示说明块"——富文本现在是采集字段，产生填报值、可作查询条件。）独占整行。

---

## 一、基础配置
| 配置项 | 说明 | 默认 |
|---|---|---|
| 题目标题 | 画布内编辑，[1,20] | 必填 |
| 是否必填 | 提交时不能为空 | 否 |
| 题目说明 | 标题下方录入指引 | 无 |
| **占位提示** | 填写者看到的空态引导（**搭建时只配提示，不在此编写正文**） | 无 |

## 二、填报与展示
1、**搭建端**：画布内仅显示带占位提示的输入区占位框，不做正文编辑。
2、**预览 / 填报**：渲染为**富文本编辑器**（B / 斜体 / 蓝 / 红 / 列表 工具条），填写者录入并排版。
3、**查看数据**：按**富文本渲染**呈现填写内容。

## 三、显示配置
| 配置项 | 值 |
|---|---|
| 布局 | 固定整行 |
| 在列表中显示 | **不支持** |

## 四、查询条件
| 配置项 | 说明 | 默认 |
|---|---|---|
| 设为查询条件 | 应用端查询抽屉出现该字段筛选 | 关 |
| 查询形式 | 模糊查询 / 精准查询 | 模糊查询 |
