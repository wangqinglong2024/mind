# 中台 · 题目属性与查询条件

> 选中题目后右栏的属性规则。含各题型属性、选项、关联症状/疾病、打分、必填，以及**新增的查询条件设置**。
> 逐题型的完整配置项另见 `../字段组件配置/`。

---

## 一、通用属性（所有题型）

1、**题目标题**：必填；未填写时保存/预览提示"标题未填写，请检查"；长度 [1,20]。
2、**必填**：勾选后填写者必须作答，标题前显示 `*`；默认不勾选。
3、**题目说明**（非必填）：标题下方的填写指引。

---

## 二、单选 / 多选

样式：**问卷网单选框 / 勾选框列表**（禁止下拉）。新加时含 标题 + 3 个默认选项。

**选项规则：**
1、每个选项内容 = **输入文字** 或 **关联"症状/疾病"**，二选一。
2、**文字输入**：非必填（保存时未填为空）；长度 [0,20]。
3、**【关联】按钮**：点开弹窗关联症状/疾病（见第五节）。
4、关联成功后：**隐藏该选项的文字输入框和【关联】按钮**，改为展示关联项 + 【x】。
   点【x】删除关联 → 恢复文字输入框和【关联】按钮（含此前临时输入的文字，前端缓存即可）。
5、每个选项**只能关联 1 个**症状/疾病。
6、点选项【X】删除该选项；点【添加选项】在题目最下方新增选项。
7、保存校验：有选项但既未填文字也未关联 → 提示"未填写或关联内容，请检查"。

注：关联的选项，后端需**记录关联关系**（症状/疾病 id），便于后期数据分析。

---

## 三、单选打分 / 多选打分

在单选 / 多选基础上，**每个选项增加分数输入框**：
1、无默认值。
2、范围 [-9999, 9999]，支持 2 位小数。
3、非必填：某选项未填分数则保存为空；填写者选中该选项时**此选项不参与计算**。

---

## 四、填空 / 单行文本 / 多行文本 / 数字 / 日期时间

1、**单行文本 / 填空**：标题 + 单行输入框（画布中输入框不可交互、仅占位）。标题必填 [1,20]。
2、**多行文本**：多行输入，独占整行。
3、**数字**：可设最小值 / 最大值 / 单位；可设为查询条件（数字范围）。
4、**日期时间**：日期 或 日期时间；可设为查询条件（时间范围）。

各题型完整项见 `../字段组件配置/`。

---

## 五、关联（症状 / 疾病）

点选项【关联】打开弹窗（自定义弹框，非浏览器原生）：

1、**说明**：症状或疾病，可任选 1 个关联，作为题目的选项。
2、**搜索症状**：数据源 = 症状基础数据公开版（已启用）；支持按**症状名称 / 拼音 / 全拼**（不区分大小写）搜索；结果显示症状名称；单选、可清除。
3、**搜索疾病**：数据源 = 疾病基础数据公开版（已启用、过滤 0 级）；支持按**疾病名称 / 拼音 / 全拼**搜索；结果显示疾病名称；单选、可清除。
4、**显示名**：非必填，长度 [0,10]。
   - 若填写 → 选项带入显示名（不带症状/疾病本名）。
   - 未填写 → 带入所选症状/疾病名称。
5、【确定】校验：
   - 同时选了症状和疾病 → 提示"症状或疾病，仅可以选择 1 个"。
   - 都没选 → 提示"未选择症状或疾病，无法关联"。

---

## 六、查询条件设置（新增，核心）

**目的**：把题目/字段变成应用端列表的筛选条件，并可把字段+规则传给算法。

**可设为查询条件的题型**：单行文本、数字、日期时间、单选、多选（后续可扩展）。

**配置项（右栏）：**
1、**设为查询条件**：开关，默认关闭。
2、**查询形式**（按题型给不同选项）：
   | 题型 | 可选查询形式 |
   |---|---|
   | 单行文本 | 模糊匹配 / 精准匹配 |
   | 数字 | 数字范围 / 精准匹配 |
   | 日期时间 | 日期时间范围 / 精准匹配 |
   | 单选 | 单选下拉筛选 |
   | 多选 | 多选筛选 |
   注：以上为部分举例，凡可查询的题型都提供该设置。
3、**在列表中显示为列**（开关，可选）：该字段是否作为应用端数据列表的一列展示。

**生效结果：**
1、应用端数据列表**上方出现对应筛选控件**（形式由查询形式决定），见业务文档《02.应用端-数据列表》。
2、被设为查询条件的**字段标识 + 查询规则**可**传给算法**做进一步计算/筛选。

注：查询条件是中台能力，具体呈现由接入的业务端渲染。

---

## 七、富文本 / 图片 / 标签文本 / 列表

- **富文本**：仅展示（说明/图文），不采集、不校验、不作查询条件。
- **图片**：上传数量上限、单张大小上限。
- **标签文本**：多个离散标签，可自由输入或选预设词，可按标签统计。
- **列表 / 子表**：可配多列（每列为非容器题型），应用端可加行/删行。
- 详见 `../字段组件配置/` 对应文件。
