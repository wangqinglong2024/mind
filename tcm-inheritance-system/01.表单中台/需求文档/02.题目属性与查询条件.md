# 中台 · 题目属性与查询条件

> 选中题目后右栏的属性规则。含各题型属性、选项、关联字段、打分、必填，以及**查询条件**与**在列表中显示**。
> 逐题型完整项另见 `../字段组件配置/`。

---

## 一、题目标题（在画布内编辑）

1、**题目标题在中间画布的题目行内直接编辑**（不在右栏属性面板）。
2、必填；未填写时保存/预览提示"标题未填写，请检查"；长度 [1,20]。

---

## 二、通用属性（右栏，所有题型）

1、**题目说明**（非必填）：标题下方的填写指引。
2、**必填**：在题目底部操作行勾选；标题前显示 `*`；默认不勾选。
3、**查询条件 / 在列表中显示**：见第六、七节（按题型是否支持）。

---

## 三、单选 / 多选（选择题）

样式：**问卷网单选框 / 勾选框列表**（禁止下拉）。新加时含 3 个默认选项。

**选项排列（属性面板）：** 单列 / 双列，切换后**中间画布联动**切换（控制应用端 1 列 / 2 列显示）。

**选项规则（画布内编辑）：**
1、每个选项内容 = **输入文字** 或 **关联字段**（症状/疾病），二选一。文字长度 [0,20]。
2、**选项关联字段**为**题目级开关**（默认关）：
   - 关闭：选项只提供文字输入。
   - 开启：每个选项出现【↔关联】，点开弹窗关联症状/疾病（见第五节）；关联成功后隐藏文字框，展示关联项 + 【x】。
3、每个选项**只能关联 1 个**症状/疾病。
4、点选项【x】删除该选项；点【＋ 添加选项】新增选项。
5、保存校验：有选项但既未填文字也未关联 → 提示"未填写或关联内容，请检查"。

**默认选项（仅单选）：** 见第四节。

注：关联的选项，后端记录关联关系（症状/疾病 id）供分析。

---

## 四、默认选项（仅单选 / 单选打分）

1、题目底部"**设置默认选项**"开关，默认关闭。
2、开启后，可在某个选项上点选"默认"，作为新建填报时的预选项（单选只能 1 个）。
3、关闭时不提示、不预选。多选 / 多选打分**不提供**默认选项。

---

## 五、单选打分 / 多选打分

在单选 / 多选基础上，**每个选项增加分数输入框**：
1、无默认值；范围 [-9999, 9999]，支持 2 位小数。
2、非必填：某选项未填分数则为空；填写者选中该选项时**此选项不参与计算**。

---

## 六、关联字段（症状 / 疾病）

点选项【↔关联】打开弹窗（自定义弹框）：

1、**说明**：症状或疾病，可任选 1 个关联，作为该选项。
2、**搜索症状**：按症状名称 / 拼音搜索、单选、可清除。
3、**搜索疾病**：按疾病名称 / 拼音搜索、单选、可清除。
4、**显示名**：非必填，[0,10]。填写则选项带入显示名；未填则带入所选症状/疾病名。
5、【确定】校验：同时选症状+疾病 → "症状或疾病，仅可以选择 1 个"；都没选 → "未选择症状或疾病，无法关联"。

---

## 七、查询条件设置

**目的**：把字段变成应用端列表的筛选条件，并可把字段+规则传给算法。

**配置项（右栏）：**
1、**设为查询条件**：开关，默认关闭。
2、**查询形式**（按题型给不同选项）：

   | 题型 | 查询形式 |
   |---|---|
   | 单选 / 多选 / 单选打分 / 多选打分 | 单选查询 / 多选查询 |
   | 单行文本 / 多行文本 / 富文本 / 标签文本 / 列表 | 模糊查询 / 精准查询 |
   | 数字 | 数字范围 |
   | 日期时间 | 日期时间范围 |
   | 图片 | 不支持查询 |

**生效结果：**
1、应用端数据列表以**右侧抽屉**呈现对应筛选（见业务《02.应用端-数据列表》）；单选查询、多选查询均为**下拉**。
2、被设查询条件的**字段标识 + 查询规则**可**传给算法**。

---

## 八、在列表中显示

1、独立开关"**在列表中显示**"，决定该字段是否作为应用端数据列表的一列。
2、**仅以下题型支持**：单选 / 单选打分 / 单行文本 / 数字 / 日期。
3、多选、多选打分、多行文本、标签文本、列表、富文本、图片**不支持**在列表中显示（属性面板不出现该开关）。

---

## 九、各题型类型属性（右栏"属性"区，摘要）

- **单行文本**：占位提示、默认值、最小/最大长度、格式校验（手机号/身份证/邮箱/纯数字）。
- **多行文本**：占位提示、默认值、最大字数、默认行数。
- **数字**：占位提示、默认值、最小/最大值、小数位数、单位。
- **日期时间**：日期精度（年月日 / 年月日时分）、默认当天。
- **图片**：数量上限、单张大小上限。
- **标签文本**：最多标签数、允许重复。
- **富文本**：**占位提示**（内容由填写者在预览/填报时录入，非搭建时编写）。
- **单选 / 单选打分**：允许清空。
- **多选 / 多选打分**：最少 / 最多选择数。
- **列表**：列配置（见 `../字段组件配置/09.列表`）。

各题型完整项见 `../字段组件配置/`。
