# 需求 · 管理端 · 预览

> 对应原型：表单配置页顶部「预览」。所见即应用端填报效果。

---

## 功能
- 点击「预览」以**全屏**方式打开当前模板的填报效果，覆盖整个窗口。
- **预览 = 应用端渲染**：使用与应用端相同的渲染逻辑，保证配置效果与实际填报一致。

## 端切换（响应式）
- 预览顶部提供 **Web 全屏 / 移动端** 两个切换按钮。
- **Web 全屏**：表单铺满可用宽度，按 Web 栅格宽度渲染。
- **移动端**：在全屏内居中显示**手机宽度（约 390px）**的手机框，内部按移动端响应式规则渲染。
- 两种模式均为全屏，仅内容宽度不同（响应式），非固定小窗。

## 交互
- 预览态可操作控件、翻页、加列表行，用于检查交互。
- 预览态点提交仅做**校验演示**，**不产生真实记录**。
- 顶部「关闭预览」退出，回到配置页。

## 规则
- 预览始终基于**当前画布中的配置**（含未保存修改），便于边配边看。
- 预览的分页、卡片、字段布局与宽度，严格遵循《07 布局与响应式规则》。
