# 业务 · 表单列表与发布

> 表单是表单中台的一个使用方（本业务里对应"问诊单"，下文统一称**表单**）。本页管理表单的搜索、筛选、发布与操作。
> 对应原型 `原型设计/admin`（列表区）。表单的搭建/题目走中台《表单搭建器》。

---

## 一、类目（左侧）

1、类目为**业务侧**的单级分类（如：冠心病、通用问诊、专家数据），与中台解耦。
2、支持 新建类目 / 重命名 / 删除；删除前若类目下仍有表单，提示先清空。
3、选中某类目 → 右侧展示该类目下的表单列表。

---

## 二、搜索与筛选（列表上方）

1、**表单标题**：按标题**模糊匹配**。
2、**发布状态**：选项 = 已发布 / 已下架 / 草稿；默认全部，可清空。
3、**创建时间**：日期**范围**筛选。

---

## 三、列表

1、**排序**：按**创建时间倒序**。
2、**列**：表单标题、发布状态、创建人、创建时间、操作。
   注：无"适用医生"列（该概念已取消）。
3、**发布状态列仅作展示**（标签：草稿 / 已发布 / 已下架），**不可点击直接切换**；发布、下架、隐藏/显示数据、删除统一在"操作"列处理。
4、右上角【＋ 创建表单】：进入中台搭建器新建（新建为"草稿"）。

---

## 四、操作（按状态区分）

### 1、草稿
- **编辑**：进入搭建器可正常编辑。
- **发布**：二次确认"确认发布该表单？发布后不可再编辑内容。"，确认后：草稿 → 已发布。
- **复制**：按当前内容带入一份相同的新草稿。
- **删除**：二次确认后删除该草稿。

### 2、已发布
- **查看**：进入搭建器**只读**，顶部提示"注：表单已发布，无法修改（仅可预览）"，仅保留【预览】与【返回列表】。
- **下架**：二次确认"下架后该表单将停止采集，且不可再重新发布。确认下架？"，确认后：已发布 → 已下架。
- **复制**：带入一份相同的新草稿。
- 注：**发布后不可编辑内容**。

### 3、已下架（禁止重新发布）
- **查看**：同已发布，进入搭建器只读、仅可预览。
- **隐藏数据 / 显示数据**（二次确认）：
  - 隐藏后，应用端数据列表**不再展示**该表单及其数据；按钮变为"显示数据"。
  - 显示后，应用端可再次查看；按钮变回"隐藏数据"。
- **复制**：带入一份相同的新草稿。
- **删除数据**：二次确认"将彻底删除该表单及其采集到的全部数据，且不可恢复。"，确认后**彻底删除**（表单 + 对应采集数据）。

---

## 五、发布状态小结

| 状态 | 列表标签 | 操作 |
|---|---|---|
| 草稿 | 草稿 | 编辑 / 发布 / 复制 / 删除 |
| 已发布 | 已发布 | 查看(只读) / 下架 / 复制 |
| 已下架·显示数据 | 已下架 | 查看(只读) / 隐藏数据 / 复制 / 删除数据 |
| 已下架·隐藏数据 | 已下架 | 查看(只读) / 显示数据 / 复制 / 删除数据 |
| 已删除 | —（记录已移除） | — |

说明：
- "已下架"标签统一显示为**已下架**，具体"显示/隐藏数据"由操作按钮体现。
- **删除为硬删除**：彻底移除表单与其采集数据，不可恢复（区别于旧版"软删"）。
- 发布状态、类目均为**业务侧**能力，与中台的表单/题目配置解耦。
