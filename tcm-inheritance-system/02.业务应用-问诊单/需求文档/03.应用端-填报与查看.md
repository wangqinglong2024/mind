# 需求 · 应用端 · 填报与查看

> 对应原型：应用端「新增填报 / 查看记录」。整页全宽，**不使用弹窗、不使用抽屉**。

---

## 进入与布局
- 点「新增填报」或记录「查看」，在**整页全屏宽度**内打开表单视图。
- 顶部为标题与「← 返回列表」，返回后回到该类目的数据列表。
- 表单按 页 → 卡片 → 字段 渲染，遵循《07 布局与响应式规则》。

## 分页填报
- 多页表单一次展示一页，顶部页指示器显示进度，可点击跳页。
- 底部「上一步 / 下一步」，最后一页显示「提交」。
- **校验时机**：点「下一步」校验当前页，未通过停留并定位到出错字段；点「提交」做全表校验。

## 各类型填报交互
- 单选 / 多选：下拉 / 多选下拉（选中项顿号连接展示）。
- 数字：只允许数值，按范围/小数位校验；带单位时右侧显示单位。
- 日期：按精度选择日期/时间。
- 标签文本：输入并回车添加标签，可逐个删除；随数量增多向下增高。
- 图片：点击上传，展示缩略图，受数量/大小限制。
- 列表（子表）：以表格填报，「添加一行」向下增行、「删除」删行；行数受最少/最多限制。

## 提交与记录
- 校验通过后提交，生成一条**记录**并回到列表，列表即时刷新出现该记录。
- 记录归属当前类目与模板，记录提交时间。

## 查看记录
- 「查看」以只读方式展示记录：字段不可编辑，隐藏提交/加行/删除等操作，保留翻页浏览。
- 只读态展示的值与填报时一致。

## 草稿 / 断点续填
- 填报中可保存草稿，下次进入可继续（正式版要求）。

## 响应式
- 填报表单随视口自适应：Web 按栅格宽度，移动端按降档规则（见《07》）。
- 列表（子表）在移动端为横向滚动表格。
